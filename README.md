# Размениум

**Размени вештина. Размени ум.**

Идејата е едноставна: наместо пари, луѓето разменуваат вештини.
Знаеш да свириш гитара? Некој друг знае да прави веб страни. Договор.

## Што може да прави

- Логирање преку Email, Google, Facebook или анонимно
- Преглед на огласи од други корисници во живо (real-time)
- Постирање сопствен оглас — нудам X, барам Y
- Измена и бришење на сопствените огласи (со потврда пред бришење)
- Омилени огласи — зачувани локално, достапни и офлајн (Room база)
- Пребарување низ огласите
- Темна / светла тема (со зачувување на изборот)
- На македонски и англиски јазик (важи на сите екрани)
- Нотификации преку Firebase Cloud Messaging
- Прилагодено и за таблет

## Архитектура

Апликацијата користи **MVVM** архитектура:

- **ViewModel** (`HomeViewModel`, `FavoritesViewModel`) — ја држи состојбата на екраните,
  преживува ротации и ги ослободува ресурсите (Firestore listener) кога не се потребни
- **Repository** (`ListingRepository`) — единствена точка за пристап до податоците
  (Firestore + Room)
- **ViewBinding** — безбеден пристап до view-ата без `findViewById`
- **Kotlin Coroutines + Flow** — асинхрони операции без блокирање на main thread
- **LiveData** — реактивно ажурирање на UI

Останати подобрувања: валидација на внесови (е-пошта, лозинка, полиња на оглас),
loading индикатори, празни состојби (empty states), локализирани пораки за грешки,
DiffUtil за ефикасно ажурирање на листата.
